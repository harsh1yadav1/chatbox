🤖 Python Chatbot (Rule-Based)

A simple rule-based chatbot built using Python.
This chatbot responds to user input using predefined keywords and random responses.

🚀 Features

💬 Interactive chat in terminal / Jupyter Notebook

🔑 Keyword-based response system

🎲 Randomized replies for natural feel

❌ Handles unknown inputs with default responses

🔚 Exit anytime using exit, quit, or bye

🛠️ Technologies Used

Python 🐍

Built-in random module

⚙️ Installation

No external libraries required ✅

Just make sure Python is installed:

python --version
▶️ How to Run
In Jupyter Notebook:

Open ChatBox.ipynb

Run all cells

Start chatting in the input prompt

In Python file:
python chatbot.py
💡 How It Works

User types a message

Program checks for keywords like:

"hello"

"how are you"

"name"

If a match is found → bot replies randomly

If no match → default response is shown
